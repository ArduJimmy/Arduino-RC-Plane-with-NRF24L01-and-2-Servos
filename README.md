# Arduino-RC-Plane-with-NRF24L01-and-2-Servos
Main components: Arduino board + NRF24L01 + 2 Servos + Brushless Motors

Please take notice that the code works at 1.8.19 Arduino IDE version and AVR Core: 1.8.5. This project (RC planes) contains 2 codes (transmitter and receiver).
Use NRF24L01 library at: https://github.com/nRF24/RF24
